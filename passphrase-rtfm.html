Syntax

function(parameter1,parameter2,parameter3);

1st parameter

A string; ElementID of input field to be monitored.

2nd parameter

A string; ElementID of filed to display password tips.

3rd parameter

A 2 digit INT; greater than 09 & less than 50, these digits dictate which meter level will be used and which character combinations are 'required' to validate.
